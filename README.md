# Portfolio Website: NATT NATTHUN

A modern, dark-themed portfolio website for a **Data Analyst & F-16 Pilot**. This project showcases data analysis expertise and professional background with a sleek, high-contrast design.

## 🚀 Objective
Build a professional digital presence hosted on GitHub Pages, featuring a modern dark theme and interactive elements.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Dark Theme), JavaScript
- **Typography:** JetBrains Mono
- **Hosting:** GitHub Pages

## 📂 Project Structure
- `index.html`: Core structure and content.
- `style.css`: Modern dark theme styling and responsive layout.
- `script.js`: Client-side interactivity.
- `assets/`: Project images and resources.
- `doc/`: Development roadmap and documentation.

## 📈 Planned Sections
- **Hero:** Introduction to NATT NATTHUN.
- **Projects:** Interactive grid showcasing data analysis work.
- **Contact:** Links to professional social profiles and email.
