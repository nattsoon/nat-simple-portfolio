# Portfolio Website Plan: NATT NATTHUN

## Objective
Create a modern, dark-themed portfolio website for a Data Analyst & F-16 Pilot, hosted on GitHub Pages.

## Tech Stack
- HTML5, CSS3 (Modern Dark Theme, JetBrains Mono font), JavaScript
- Hosting: GitHub Pages

## Project Structure
- `index.html`: Main structure
- `style.css`: Dark theme styling, typography, responsive layout
- `script.js`: Interactivity
- `assets/`: Images or additional resources

## Implementation Steps
1. **Initialize Project**: Create directory structure and basic `index.html`.
2. **Design & CSS**:
   - Apply JetBrains Mono font (via Google Fonts or local import).
   - Implement "Modern Dark" color palette (slate-grey/charcoal backgrounds, neon or high-contrast accent colors).
3. **Sections Construction**:
   - **Hero**: Introduce "NATT NATTHUN" with reference to Data Analysis & F-16 piloting background.
   - **Projects**: Grid layout for 3 data analysis projects.
   - **Contact**: Section with social links/email.
4. **GitHub Deployment**:
   - Initialize git, commit, and push to GitHub.
   - Enable GitHub Pages in repository settings.

## Verification & Testing
- Validate mobile responsiveness.
- Check font rendering and dark mode color consistency.
- Verify GitHub Pages deployment link.
